# Skill Repo Tracker v1.1.9

## 中文

v1.1.9 让 GitHub、仓库、Skill 和插件列表更适合日常整理：更稳定的默认排序、更宽的搜索入口，以及能帮助回看来源的时间列。

### 更新亮点

- 四个主要列表默认按名称 A-Z 排序，并保持大写优先，打开页面后更容易扫读和定位。
- GitHub 和仓库页的搜索框现在可以同时搜索名称、备注和 README 内容，适合按“用途”或“记忆片段”找回仓库。
- Skill 页可以搜索备注、SKILL.md 内容和来源仓库 README；插件页可以搜索备注和来源摘录，减少来回点详情的次数。
- GitHub 页新增 Star 时间，仓库、Skill 和插件页新增添加时间，并支持按时间排序，方便回看收藏和收录顺序。
- GitHub 详情快捷操作新增复制仓库链接，分享、记录或粘贴到其他工具更顺手。
- 新增项目级 `AGENTS.md`，把 Codex 协作规则、宣传素材边界和本地服务约定固化到仓库里。

## English

v1.1.9 makes the GitHub, repository, Skill, and plugin lists easier to use for
daily organization: steadier default sorting, broader search, and timestamps
that help you understand where an item came from and when it was collected.

### Highlights

- The four main lists now default to A-Z name sorting with uppercase names
  first, making each page easier to scan as soon as it opens.
- GitHub and repository search can now match names, notes, and README content,
  so you can find repositories by purpose or remembered context.
- Skill search can match notes, SKILL.md content, and source repository README
  text; plugin search can match notes and source excerpts, reducing the need to
  open details repeatedly.
- GitHub repositories now show Starred at, while repositories, Skills, and
  plugins show added timestamps. These columns can be sorted to review starring
  and collection order.
- GitHub details now include a quick action to copy the repository URL, making
  sharing and note-taking smoother.
- A project-level `AGENTS.md` now documents Codex collaboration rules, promo
  material boundaries, and local service conventions in the repository.

### macOS DMG

The attached DMG is an Apple Silicon macOS local installation package signed
with ad-hoc signing. It is suitable for zero-cost testing and sharing through
GitHub Releases.

It is not Developer ID signed or Apple notarized. On first launch, macOS may ask
you to manually allow the app.

Asset:

- GitHub Release asset: `Skill.Repo.Tracker_1.1.9_aarch64.dmg`
- Local build file: `Skill Repo Tracker_1.1.9_aarch64.dmg`

SHA-256:

```text
8a29fb4e696cc99f338d01d0af31e384b585e1df32518900093c58553647e46a
```

Install notes:

1. Download and open `Skill.Repo.Tracker_1.1.9_aarch64.dmg`.
2. Drag `Skill Repo Tracker.app` into `/Applications`.
3. On first launch, macOS may block the app because it is ad-hoc signed. Control-click the app in Finder, choose Open, then confirm Open.
4. If it is still blocked, open System Settings -> Privacy & Security and choose Open Anyway for Skill Repo Tracker.
5. Technical users can clear the quarantine attribute:

```bash
xattr -cr "/Applications/Skill Repo Tracker.app"
```

Safety boundaries:

- GitHub tokens are never included in migration packages. Add your token again
  on each new machine.
- This DMG is an ad-hoc test package, not a Developer ID notarized public
  installer.
