# Skill Repo Tracker v1.1.8

## 中文

v1.1.8 增加了本地备注和 JSON 迁移包，用来管理 GitHub、仓库、Skill、插件入口和备注数据。

### 更新亮点

- GitHub 仓库、已追踪仓库、Skill、插件入口都可以记录本地备注。
- GitHub 页和仓库页里的同一个 GitHub 仓库共用一条备注，只需要编辑一次。
- 设置页支持导出/导入 JSON 迁移包，包含 GitHub 元数据、仓库、Skill、插件和备注。
- 迁移包不会导出 GitHub token、Keychain 密钥、本地 Skill 文件、源码 ZIP 或任务日志。

### macOS DMG

本次 Release 附带的是 Apple Silicon macOS 本地安装测试包，使用 ad-hoc 自签名，适合零成本测试和分享。

它不是 Developer ID 签名和 Apple notarized 的公开免提醒安装包。首次打开时，macOS 可能要求你手动放行。

下载资产：

- GitHub Release 文件名：`Skill.Repo.Tracker_1.1.8_aarch64.dmg`
- 本地构建文件名：`Skill Repo Tracker_1.1.8_aarch64.dmg`

SHA-256：

```text
72df3efdb9fb9a490685050bebb6c14796922d3ce4088aa8dccdb9f2bdf7a8b7
```

安装注意事项：

1. 下载并双击打开 `Skill.Repo.Tracker_1.1.8_aarch64.dmg`。
2. 把 `Skill Repo Tracker.app` 拖入 `/Applications`。
3. 首次启动如果提示“无法验证开发者”，请在 Finder 里右键 App，选择“打开”，再确认“打开”。
4. 如果右键打开仍被拦截，请进入“系统设置 -> 隐私与安全性”，在底部找到 Skill Repo Tracker 并点击“仍要打开”。

```bash

```

安全边界：

- 这个 DMG 是 ad-hoc 测试包，不等同于 Developer ID notarized 的公开发行包。

## English

v1.1.8 adds local notes and JSON migration packages for the GitHub, repository,
Skill, and plugin workflows.

### Highlights

- Notes for GitHub repositories, tracked repositories, Skills, and plugin
  entries.
- Shared note identity for the same GitHub repository across the GitHub and
  repository pages.
- JSON migration package export/import for GitHub metadata, repositories,
  Skills, plugins, and notes.
- Migration packages do not export GitHub tokens, Keychain secrets, local Skill
  files, source ZIPs, or task logs.

### macOS DMG

The attached DMG is an Apple Silicon macOS local installation package signed
with ad-hoc signing. It is suitable for zero-cost testing and sharing through
GitHub Releases.

It is not Developer ID signed or Apple notarized. On first launch, macOS may ask
you to manually allow the app.

Asset:

- GitHub Release asset: `Skill.Repo.Tracker_1.1.8_aarch64.dmg`
- Local build file: `Skill Repo Tracker_1.1.8_aarch64.dmg`

SHA-256:

```text
72df3efdb9fb9a490685050bebb6c14796922d3ce4088aa8dccdb9f2bdf7a8b7
```

Install notes:

1. Download and open `Skill.Repo.Tracker_1.1.8_aarch64.dmg`.
2. Drag `Skill Repo Tracker.app` into `/Applications`.
3. On first launch, macOS may block the app because it is ad-hoc signed. Control-click the app in Finder, choose Open, then confirm Open.
4. If it is still blocked, open System Settings -> Privacy & Security and choose Open Anyway for Skill Repo Tracker.
5. Technical users can clear the quarantine attribute:

```bash
xattr -cr "/Applications/Skill Repo Tracker.app"
```

Safety boundaries:

- GitHub tokens are never included in migration packages. Add your token again
  on each new machine.
- This DMG is an ad-hoc test package, not a Developer ID notarized public
  installer.
