# Skill Repo Tracker v1.1.10

## 中文

v1.1.10 把四个主要列表的搜索体验拆得更清楚：主搜索只按名称定位，内容搜索专门查备注、README、SKILL.md 或插件摘录。大量仓库、Skill 和插件混在一起整理时，结果更稳定，也更少误命中。

### 更新亮点

- GitHub、仓库、Skill 和插件页的主搜索现在只搜索名称，让“按名字找东西”更直接、更可预期。
- GitHub 和仓库页新增独立内容搜索，可以单独搜索备注和 README，适合按用途、场景或记忆片段找回条目。
- Skill 页新增独立内容搜索，覆盖备注、`SKILL.md` 和来源仓库 README；查找时优先使用个人备注，其次是 Skill 自身内容，最后才是来源仓库 README。
- 插件页新增独立内容搜索，可以搜索备注和来源摘录，方便从安装入口或记录片段定位插件。
- 名称搜索和内容搜索可以同时使用，适合先按名字缩小范围，再按备注或文档内容精确定位。

## English

v1.1.10 makes search clearer across the four main lists: the primary search now
finds items by name only, while content search is dedicated to notes, README,
SKILL.md, or plugin source excerpts. When you manage a large collection of
repositories, Skills, and plugins, results are steadier and less noisy.

### Highlights

- The GitHub, repository, Skill, and plugin pages now use name-only primary
  search, making direct lookup more predictable.
- GitHub and repository pages now have a separate content search for notes and
  README text, so you can find items by purpose, context, or remembered details.
- The Skill page now has a separate content search across notes, `SKILL.md`, and
  source repository README text. It prioritizes personal notes first, then the
  Skill's own content, then the source repository README.
- The plugin page now has a separate content search for notes and source
  excerpts, making plugin entries easier to find from install snippets or saved
  context.
- Name search and content search work together, so you can narrow by name first
  and then refine by notes or documentation content.

### macOS DMG

The attached DMG is an Apple Silicon macOS local installation package signed
with ad-hoc signing. It is suitable for zero-cost testing and sharing through
GitHub Releases.

It is not Developer ID signed or Apple notarized. On first launch, macOS may ask
you to manually allow the app.

Asset:

- GitHub Release asset: `Skill.Repo.Tracker_1.1.10_aarch64.dmg`
- Local build file: `Skill Repo Tracker_1.1.10_aarch64.dmg`

SHA-256:

```text
82198391710320eda34f824fb29d3a946d843e6f7bc3d74d4041a5c3aed90a70
```

Install notes:

1. Download and open `Skill.Repo.Tracker_1.1.10_aarch64.dmg`.
2. Drag `Skill Repo Tracker.app` into `/Applications`.
3. On first launch, macOS may block the app because it is ad-hoc signed. Control-click the app in Finder, choose Open, then confirm Open.
4. If it is still blocked, open System Settings -> Privacy & Security and choose Open Anyway for Skill Repo Tracker.
5. Technical users can clear the quarantine attribute:

```bash
xattr -cr "/Applications/Skill Repo Tracker.app"
```

Safety boundaries:

- GitHub tokens are never included in migration packages. Add your token again
  on each new machine.
- This DMG is an ad-hoc test package, not a Developer ID notarized public
  installer.
