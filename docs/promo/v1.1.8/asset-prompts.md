# Skill Repo Tracker v1.1.8 Image Prompt Notes

These assets were regenerated with the `ian-xiaohei-illustrations` skill after
rejecting the initial script-drawn placeholder images.

## Style Contract

- Pure white background, black hand-drawn line art, lots of blank space.
- 小黑 must perform the core action, not stand beside the diagram.
- Sparse red/orange/blue handwritten annotations only.
- No PPT infographic, formal flowchart, gradient, shadow, cute mascot, or app UI
  screenshot look.

## Final Illustration Themes

- `01-source-fog-records.png`: source fog pulled into trackable records.
- `02-repository-archive-cabinet.png`: repositories as rollback-capable archive
  drawers with SHA, ZIP, manifest, and recognition result.
- `03-skill-library-publish.png`: one local Skill library publishing outward to
  tool directories without write-back.
- `04-plugin-entry-index.png`: scattered install commands clipped into a plugin
  entry index, with no automatic installation.
- `05-migration-box-token-safe.png`: metadata and notes move in a migration box
  while token and Keychain secrets stay behind.
- `wechat-cover.png`: local workbench cover, 900x383, with GitHub, Repos,
  Skills, Plugins, notes, backup, and migration cues.
