# Skill Repo Tracker v1.1.6 Promotion Kit

This folder contains the public-facing promotion materials for Skill Repo Tracker v1.1.6.

## Core Message

Skill Repo Tracker turns scattered AI Skill repositories and plugin install entries into a local workbench where users can see source, version, path, publish targets, backup records, and risk boundaries before acting.

Important boundary: v1.1.6 recognizes common plugin install entries and links them to repositories/Skills. It is not a full plugin marketplace, does not run plugin installation, and does not guarantee third-party entry safety.

## Contents

- `wechat-article.md`: WeChat article draft in Simplified Chinese.
- `article-assets/`: article images and current v1.1.6 software screenshots.
- `video-script.md`: shared script for both video versions.

Local validation builds are not notarized. Do not describe them as public signed DMG releases unless notarization is completed.
