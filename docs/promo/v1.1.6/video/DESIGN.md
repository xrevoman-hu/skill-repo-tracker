# Skill Repo Tracker Promo Design

## Style Prompt

A precise local developer workbench video: quiet confidence, visible structure, and small safety mechanisms made tangible. The product is about provenance, sync boundaries, backups, and trust. Use a dark graphite canvas with warm paper text and a single amber-green accent, avoiding generic neon tech styling. The motion should feel like files becoming organized into a stable local library.

## Colors

- Background: `#111513`
- Panel: `#18201C`
- Panel light: `#25302A`
- Text: `#F4EFE3`
- Muted text: `#B9C0B2`
- Accent: `#D8A84F`
- Safety green: `#7FB685`
- Warning red: `#D76D57`

## Typography

- Display: system `sans-serif`, 800-900, for large idea statements.
- Body and labels: system `sans-serif`, 400-700, for precise product language.
- Data and paths: system `monospace`, 500-700, for paths, version labels, and target names.

## Motion

- Use medium-energy product explainer pacing.
- Primary transition: warm push slide or soft blur crossfade.
- Entrance animations should combine opacity, y movement, and subtle scale.
- Background particles and file chips drift slowly but deterministically.
- Never use hard jump cuts.

## What NOT to Do

- Do not use blue-purple gradients, generic neon grids, or glowing cyberpunk cards.
- Do not turn the piece into a landing page hero.
- Do not overcrowd the frame with tiny UI text.
- Do not imply notarized public distribution; keep the artifact status honest.
