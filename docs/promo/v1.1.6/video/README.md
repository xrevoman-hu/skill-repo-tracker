# HyperFrames Promo Video

This is a vertical 1080x1920 HyperFrames composition for the Skill Repo Tracker v1.1.6 promotional video.

It uses the shared narration in `narration.txt` and the current v1.1.6 screenshot assets. The output should describe plugin install-entry recognition as indexing and inspection, not automatic plugin installation.

## Commands

```bash
cd docs/promo/v1.1.6/video
npx hyperframes lint
npx hyperframes inspect
npx hyperframes render --output skill-repo-tracker-v1.1.6-hyperframes.mp4 --quality standard
```
