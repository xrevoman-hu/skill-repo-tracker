# Skill Repo Tracker v1.1.6 视频对比

本次两版视频使用同一份宣传主线、同一份旁白和同一批核心素材：焦虑来自看不清 Skill/插件入口的来源和影响边界，价值来自看清、备份、再行动。

## 产物

- HyperFrames: `docs/promo/v1.1.6/video/skill-repo-tracker-v1.1.6-hyperframes.mp4`
- Remotion: `docs/promo/v1.1.6/video-remotion/out/skill-repo-tracker-v1.1.6-remotion.mp4`
- 旁白: `docs/promo/v1.1.6/video/assets/narration.wav`

## 对比结论

| 维度 | HyperFrames 版 | Remotion 版 |
| --- | --- | --- |
| 适合场景 | 更适合快速做产品宣发短片，HTML/CSS 调整直观 | 更适合可代码化复用的长期视频模板 |
| 视觉气质 | 更接近产品说明页，节奏稳，截图和大字信息切换直接 | 更像品牌化动态海报，分镜结构更强 |
| 验证方式 | `lint`、`inspect`、`render` 已跑通 | still 布局检查和 `render` 已跑通 |
| 风险 | 依赖页面结构和 CSS 约束，复杂交互镜头需要更多手动检查 | 初始搭建成本更高，浏览器/依赖下载对环境更敏感 |

## 推荐使用

公众号和朋友圈首发建议使用 HyperFrames 版：它更接近这次 v1.1.6 的“看清楚再行动”叙事，产品截图的存在感更强。

如果后续要做连续版本更新视频，建议把 Remotion 版沉淀成长期模板：版本号、功能点、截图和旁白都可以结构化替换。
